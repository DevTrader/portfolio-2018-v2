Rocket Finder

Live Demo | Github

Project Description:

Rocket Finder is a pure front-end project taking advantage of AJAX to fetch information from launches on launchlibrary.net and WebGL Earth API to render a 3D earth, where I pin the positions of the launches. It features an animation algorithm to create a "camera path" where the selected rocket launch comes into focus on the 3D globe.

Technologies used: HTML, CSS, jQuery, AJAX and WebGL

Ravens

Live Demo | Github

Project Description:

Ravens is an experimental chatting app, taking advantage of Socket.io, in this app the chats have a life-time of 15 minutes (set to accomodate herokus 30 minute limits). This app was my way of exploring Socket.io, Three.js (which is rendering the birds on the landing-page) and some fun algorithms (the name randomizer for example).

Technologies used: Socket.io, Three.js, Node, Express and MongoDB

Cryptofolio

Live Demo | Github

Project Description:

Cryptofolio was created during the cryptocurrency craze as a way to help users track multiple digital currencies. It features a high-performance algorithm that generates its own unique historical data for each user. At the moment the amount of supported currencies is capped at 100 to save on database space (since my algorithm is also saving historical data of them on a daily schedule).

Technologies used: React, Node, MongoDB and Express
